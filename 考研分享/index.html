    # 备考

    # 政治

    政治这门课呢算是道路最清晰透明的，简单概括即：背！

    政治正确,拥护党的领导->不及格都难。

    学习大纲知识点->刷选择题->学习时政->刷选择题->做模拟卷->背诵押题讲义（大题）

    时间（月份）事件视频书籍占用时间注
    7、8、9徐涛强化班视频、刷1000题徐涛强化班《肖秀荣1000题》晚上1~2小时可以看完某一章视频或书后立刻做对应选择以加深印象，空闲时可再看看错题
    10复习1000题，背诵重点如党的会议时间线、马原知识梳理图等。腿姐强化班（看的不多）《徐涛背诵手册》《肖秀荣1000题》晚上1~2小时
    11隔天做一套肖8，腿4，结合背诵手册背诵重点事件。《肖秀荣八套卷》《腿姐四套卷》晚上1~2小时仅做了选择题
    12肖4：完成选择，熟练背诵大题答案（即押题）；复习肖8选择题；熟练背诵腿姐押题讲义。腿姐冲刺班《肖秀荣四套卷》《肖秀荣八套卷》《腿姐押题讲义》晚上3-4小时视频(技巧or心理安慰

    最后71。。不求有功但求无过

    # 英语

    时间（月份）事件视频书籍/工具事件备注
    3看田静长难句视频，背《恋恋有词》田静长难句《恋恋有词》晚上1~2小时田静长难句对于基本句型讲的比较细，看完后对于长难句的切分理解了个大概，视频不长大概一周看完。《恋恋有词》一天一个单元正好一个月。
    4-12滚动做英语一英语二真题的客观题部分。背单词。唐迟阅读的逻辑（看的不多）《张剑英语真题》、墨墨背单词真题下午1~2小时、单词晚上1小时英语真题刷了两遍，留了一、二各一套用于考前模拟。墨墨背单词拥有载入各家单词书的功能（我载入了恋恋有词和红宝书），结合艾宾浩斯遗忘曲线效果不错。
    11学习翻译、作文刘晓燕作文、翻译刘晓燕作文讲义、《高分写作》晚上1~2小时王江涛等老师推荐背整篇作文，刘晓燕潘赟等老师推荐掌握各种组合句型拼出作文，按自身情况选择即可。可能没耐心没方法？翻译完全凭感觉发挥。。
    12背作文、试写作文刘晓燕作文讲义、《高分写作》晚上1~2小时作文评分：老师、微软爱写作、Grammarly。我认为再较少时间内拿到平均分性价比高（高玩除外），最后考试中两篇作文占用45分钟。

    # 数学

    这个能说的就比较多了，

    时间（月份）事件视频书籍/工具时间备注
    3-5看汤家凤高数基础等视频，写1800基础题。汤家凤高数基础、寒假直播、专题直播等。（当时各种直播都会在B站发布）《汤家凤1800题》下午or上午3小时。我先买的是数一版本，后来当作数二版本使用。当时正值疫情在家摸鱼时期，基本得靠去图书馆才有的写，故写的很慢。
    6看汤家凤线代基础，写1800基础题。汤家凤线代基础、专题直播等。（当时各种直播都会在B站发布）《汤家凤1800题》下午or上午4~5小时。返校了！！！尽管早上仍起不来，但学习时间确实增加了。
    7月上旬汤家凤高数强化，写1800提高题。汤家凤高数强化《汤家凤1800题》共6-7小时。穿插着回顾基础题错题（随便看看不花多少时间），二刷疑难点。1800中尤其是积分不等式那里有一些偏题怪题，可先跳过，不影响心态。
    7月下旬刷张宇闭关修炼、李林108部分困惑点刷了张宇对应的视频。《闭关修炼》《李林108题》共6-7小时。这两本书题目并不是很多，所以两周足矣，同时这两本书也不是很大众化，可以选择其他资料。我没有写汤1800的线代强化，在这两本书中得到一定程度的弥补。
    8月上旬学习张宇高数18讲，二刷闭关修炼、李林108高数部分。张宇高数18讲视频《张宇高数18讲》《闭关修炼》《李林108题》共6-7小时。18讲确实是一本对提高很有用的书，知识结构图清晰易懂，题目思维性很强、覆盖广，后面的题目可以选做一些。二刷闭关、108主要是其他也没啥资料了，有些题值得再看一下。
    8月下旬张宇线代9讲，二刷闭关修炼、李林108线代部分。张宇线代9讲视频《张宇线代9讲》《闭关修炼》《李林108题》共6-7小时。线代9讲评价同上，难度个人认为比李永乐线代讲义高上一截。这段时间也找了些碎片时间再看看汤1800错题。
    92010-2020真题《张宇真题大全》上午3-4小时基本按照考试时间8:30-11:30完成，有时起不来拖到9:00-12:00。做了那么多练习，真题题型基本熟悉，原题可能也做过不少，主要就培养个套卷习惯，同时减少粗心。
    10-12数学试题结构改革，做模拟卷武忠祥17堂课（看了部分）、张宇国庆特训、张宇冲刺课《张宇8、4套卷》《李林6、4套卷》《超越5套卷》上午3-4小时这段期间也没什么新知识需要系统学习了，主要还是做模拟卷提升手感，减少粗心！！！同时有时间就复习错题。
